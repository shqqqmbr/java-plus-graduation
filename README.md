# KV Service

gRPC сервис для работы с ключ-значение хранилищем на базе Tarantool.

## Технологии

- Java 21
- gRPC
- Tarantool 3.2.x
- Maven
- Docker

## API

| Метод | Описание |
|-------|----------|
| `Put` | Сохранить или перезаписать значение по ключу |
| `Get` | Получить значение по ключу |
| `Delete` | Удалить значение по ключу |
| `Range` | Получить stream пар ключ-значение в диапазоне |
| `Count` | Получить количество записей |
